# ClipJolt Homepage Design Specifications

Based on the reference website (SaaSWiftie), here are the key design elements to replicate:

## Design System

### Typography
- **Primary Font**: Inter (modern, clean sans-serif)
- **Heading Sizes**: Large, bold headings for hero section
- **Body Text**: Regular weight, good readability
- **Font Weights**: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

### Color Palette
- **Background**: White (#FFFFFF)
- **Text Primary**: Black/Dark Gray (#000000 or #1a1a1a)
- **Accent Blue**: Bright blue for CTAs (#2563EB or similar)
- **Secondary Colors**: Light gray backgrounds for sections (#F9FAFB)
- **Badge Colors**: Yellow (#FCD34D), Blue (#3B82F6), Green (#10B981), Pink (#EC4899), Purple (#8B5CF6)

### Layout Structure

1. **Navigation Bar**
   - Clean, minimal header
   - Logo on left
   - Navigation links centered/right
   - CTA button (blue, rounded)
   - White background with subtle border

2. **Hero Section**
   - Centered layout
   - Small badge/tag at top (with member count)
   - Large heading (2-3 lines)
   - Subtitle/description text
   - Two CTA buttons (primary blue, secondary outline)
   - Logo cloud/partner logos below

3. **Feature/Product Showcase**
   - Large product screenshot or mockup
   - Clean white card with shadow
   - Shows actual product interface

4. **Benefits Section**
   - Black badge label
   - Heading + description
   - Icon-based feature list
   - Blue checkmark or star icons
   - Left-aligned text

5. **Interactive Elements Section**
   - Colorful pill-shaped buttons/tags
   - Gradient backgrounds on some elements
   - Playful, modern aesthetic

6. **Stats/Metrics Section**
   - Black badge label
   - Large numbers with context
   - Chart/graph visualization
   - Clean, data-driven design

7. **Feature Cards**
   - Two-column layout
   - Icon + heading + description
   - Blue accent icons
   - White background cards

8. **Testimonials/Team Section**
   - Avatar images
   - Names and roles
   - Clean, minimal presentation

9. **Footer**
   - Simple, minimal
   - "Made in Figma" or similar tagline
   - Centered text

## Component Patterns

### Buttons
- **Primary**: Blue background, white text, rounded corners
- **Secondary**: White background, border, black text
- **Size**: Medium padding, comfortable click targets

### Badges
- Small, pill-shaped
- Various colors for different purposes
- Subtle or bright depending on context

### Cards
- White background
- Subtle shadow
- Rounded corners
- Good padding/spacing

### Icons
- Simple, clean line icons
- Blue accent color
- Consistent size and style

## Spacing System
- Generous whitespace
- Consistent padding (16px, 24px, 32px, 48px, 64px)
- Section spacing: 80-120px between major sections

## Responsive Considerations
- Mobile-first approach
- Stack columns on smaller screens
- Maintain readability and touch targets
- Adjust font sizes appropriately
